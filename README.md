## Try It Out: 

Prerequisites:
- Python
- Node.js & npm
- A Google AI Studio API Key

Clone the repo:
<pre>
git clone https://github.com/yourusername/Wise-Whale-Chatbot.git
</pre>

Launch the Backend:
<pre>
cd flask-server
pip install -r requirements.txt
# Create a .env file and add: GEMINI_API_KEY=your_key_here
python flask-server.py
</pre>
  
Launch the Frontend:
<pre>
cd ../client
npm install
npm run dev
</pre>
